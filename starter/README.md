# GoIgniter Starter

A minimal starter template for GoIgniter framework.

## Quick Start

```bash
# Install dependencies
go mod tidy

# Run
go run main.go
```

Open http://localhost:8080

## Structure

```
├── application/
│   ├── controllers/    # Your controllers
│   └── views/          # HTML templates
├── public/             # Static files
├── go.mod
└── main.go             # Entry point
```
