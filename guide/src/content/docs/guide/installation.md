Goigniter membutuhkan **Go 1.20** atau yang lebih baru.

## Via Go Get

Cara termudah untuk memulai adalah menggunakan `go get`:

```bash
go get [github.com/semutdev/goigniter](https://github.com/semutdev/goigniter)
