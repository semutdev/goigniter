## Mengapa Goigniter?

<CardGrid stagger>
	<Card title="Familiar Syntax" icon="pencil">
		Struktur folder `controllers`, `models`, `views` yang sama persis dengan CI3. Gak perlu adaptasi lama.
	</Card>
	<Card title="Golang Performance" icon="rocket">
		Dapatkan kecepatan *compiled language*. Request selesai dalam hitungan mikrodetik, bukan milidetik.
	</Card>
	<Card title="Auto Routing" icon="setting">
		Magic Routing ala CI3. Cukup buat file controller, method `Index()` otomatis jadi root.
	</Card>
</CardGrid>
